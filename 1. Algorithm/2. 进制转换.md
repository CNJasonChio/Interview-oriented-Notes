### 1. 进制转换

- 进制转换都是以进制为底，进行循环除的操作。
- 当进制中出现非字母时，可以用表（形式不限）和进制中的数一一对应
- 循环之后，需要翻转一下才能得到正确答案

```c++
string toHex(int num) {
    string map = "0123456789abcdef";
    string result;
    unsigned int tmp = num;
    while (tmp){
        result.push_back(map[tmp % 16]);
        tmp /= 16;
    }
    reverse(result.begin(), result.end());
    return result;
}
```
